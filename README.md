# NginX Redirect Writer CLI

Turn a CSV of new and old URLs into NginX rewrite rules.

## Installation

```
$ npm install -g nginx-redirect-writer-cli
```

## Usage

```
$ nrw <input.csv> <output.conf>
$
$ # Example:
$ nrw my-redirects.csv my-redirects.conf
```
